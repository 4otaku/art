Для создания нового баллона с переводом, кликните на любое свободное место изображения.
Для того чтобы изменить размер баллона, потяните за его правый или нижний край, либо за правый-нижний угол.
Для перемещения баллона используйте drag-n-drop, то есть кликните на нем левой кнопкой мыши и не отпуская ее перенесите баллон на новое место.
Для редактирования текста баллона используйте на нем двойной клик.
